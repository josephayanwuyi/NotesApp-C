# NotesApp-C

Command-line Notes Application written in C.

## Features
- Add notes
- List notes
- Delete notes
- Timestamped entries
- Input validation
- Clear-screen text interface

## How to Compile and Run

```bash
clang -o notes main.c -lcs50
./notes


Project Structure
main.c — Core application logic
notes.txt — Stores notes (generated at runtime)
